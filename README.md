# Adaptation of Cobalt 2 theme for [Vim Airline Themes]()

Based on the [Cobalt2 theme by Wes Bos](https://github.com/wesbos/cobalt2/).
To install, add the cobalt2.vim file inside the vim-airline-theme `themes` folder
